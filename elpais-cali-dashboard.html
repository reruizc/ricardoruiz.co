Estado del backend (mayo 2026 · borrador v1):
Los KPIs y el stream en vivo están conectados a un mock de datos para diseño. Para conectar a producción se necesita una Lambda agregadora que lea el cache de DeepSeek (s3://elecciones-2026/ricardoruiz.co/test-presidencial-2026/cache/) y emita JSON con conteos por candidato / arquetipo / barrio del Valle.
