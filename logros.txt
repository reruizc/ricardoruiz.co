# Cada logro se evalúa sobre el total de votos del candidato (no sobre el universo).

BALUARTE DEPARTAMENTAL
  Condición : Un solo departamento concentra > 60% del total de votos del candidato.
  CSS class : dep (verde)
  Ejemplo   : "BALUARTE DEPARTAMENTAL · Antioquia — 73%"

BALUARTE REGIONAL
  Condición : Dos departamentos juntos concentran > 80% del total de votos del candidato.
  CSS class : reg (morado)
  Ejemplo   : "BALUARTE REGIONAL · Antioquia & Bogotá D.C. — 85%"
